# 高级策略示例

本目录包含三个高级策略示例，展示 aquant 的实战应用。

## 策略列表

### 1. 多因子选股策略 (`multi_factor_strategy.py`)

结合多个因子进行股票选择：

- **动量因子**：过去 20 日收益率，捕捉价格趋势
- **反转因子**：过去 5 日收益率的负值，捕捉短期反转
- **成交量因子**：相对平均成交量，判断资金活跃度

策略逻辑：
1. 对股票池中每只股票计算三个因子得分
2. 按权重合成总得分（动量 0.4，反转 0.3，成交量 0.3）
3. 选出得分最高的前 N 只股票
4. 等权重持仓

适用场景：中长期持仓，适合大盘股选股

### 2. 配对交易策略 (`pairs_trading_strategy.py`)

基于两只相关股票的价差进行交易：

- 计算价差的历史均值和标准差
- 当价差偏离均值超过 2 倍标准差时开仓
- 当价差回归至 0.5 倍标准差内时平仓

策略逻辑：
1. 价差过高：做空价差（卖 A 买 B）
2. 价差过低：做多价差（买 A 卖 B）
3. 价差回归：平仓

适用场景：市场中性策略，适合行业内相关度高的股票对

### 3. 均值回归策略 (`mean_reversion_strategy.py`)

基于布林带的均值回归策略：

- 使用 20 日移动平均线作为中轨
- 上轨 = 中轨 + 2 倍标准差
- 下轨 = 中轨 - 2 倍标准差

策略逻辑：
1. 价格突破上轨：做空（预期回归）
2. 价格突破下轨：做多（预期回归）
3. 价格回归中轨：平仓

适用场景：震荡市场，适合波动率较高的股票

## 运行示例

```bash
# 多因子选股策略
uv run python examples/multi_factor_strategy.py

# 配对交易策略
uv run python examples/pairs_trading_strategy.py

# 均值回归策略
uv run python examples/mean_reversion_strategy.py
```

每个策略运行后会生成 HTML 报告，可在浏览器中查看详细结果。

## 学习建议

1. **从简单到复杂**：先理解均值回归策略的逻辑，再学习配对交易和多因子
2. **参数调优**：尝试修改策略参数（如回溯期、阈值），观察对结果的影响
3. **风险管理**：在实际应用中需添加止损、仓位控制等风控措施
4. **数据质量**：确保使用高质量的历史数据，避免前视偏差

## 扩展方向

- 实现动态仓位管理
